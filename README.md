# Google Play Market Analysis & Insights

## Overview
Data-driven analysis of Google Play Store apps examining pricing, ratings, installs, category saturation, and user sentiment.

## Tools Used
- Excel (Data Cleaning)
- Tableau (Visualization)
- Kaggle Dataset

## Key Findings
- Free apps dominate installs.
- Paid apps have slightly higher ratings.
- Ratings positively correlate with installs.
- Category competition impacts visibility.
